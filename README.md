#  Notes App :

This is a simple and responsive **Notes Application** built using HTML, CSS, and JavaScript.  
Users can add notes, view them, and delete them.  
All notes are stored in localStorage, so the data remains even after refreshing the page.

---

##  Features :
- Add new notes  
- Display all saved notes  
- Delete notes  
- Notes saved in localStorage  
- Data not lost after page refresh  
- Fully Responsive (Mobile + Tablet)

---

##  How to Run the Project :
1. Download or clone the repository  
2. Open the folder  
3. Double-click  index.html 
4. The app will open in a browser  
5. Start adding notes!
